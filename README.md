##  Descripción del Proyecto

Este repositorio contiene el código fuente, diseño y documentación técnica del proyecto **"Ortesis robótica para asistencia del movimiento de coxofemoral y rodilla en adultos con hemiplejia derecha"**.

El objetivo principal es desarrollar un dispositivo mecatrónico capaz de asistir en la rehabilitación física, proporcionando movimientos controlados y seguros para pacientes con movilidad reducida en el miembro inferior derecho. El sistema integra diseño mecánico, control electrónico e interfaces de usuario para su operación.

## Tecnologías Utilizadas

* **Software de Control:** Python (con PyQt/Tkinter para la interfaz gráfica).
* **Documentación:** LaTeX (Clase personalizada UPIITA).
* **Hardware:** Motores Nema, Drivers, Sensores, Actuadores Lineales.

## Instalación y Ejecución

### Requisitos Previos
Asegúrate de tener instalado Python 3.8 o superior.

### Clonar el repositorio
```bash
git clone https://github.com/ramsalue/ortesis-robotica
cd ortesis-robotica
```

### Ejecutar la Interfaz de Control

```bash
python app_fisioterapia.py
```
